## User Roles

The Kavach system supports three main user roles:

### Customer
Customers are end users who access the platform to utilize services. They can:
- Create and manage their profile
- Access customer-specific features
- Manage their account settings and security

### Expert
Experts are service providers on the platform. They can:
- Create detailed professional profiles
- Manage their expertise and qualifications
- Access expert-specific tools and features

### Administrator
Administrators manage the platform and users. They can:
- Manage user accounts and permissions
- Access administrative tools and reports
- Configure system settings
