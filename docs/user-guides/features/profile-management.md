# Profile Management

This guide covers how to manage your profile information, including personal details, professional information, and profile settings.

## Accessing Your Profile

### Viewing Your Profile
1. Click your profile picture or name in the top navigation
2. Select "Profile" from the dropdown menu
3. Or navigate directly to `/profile`

### Profile Overview
Your profile page displays:
- Basic personal information
- Professional details (for experts)
- Account status and verification badges
- Profile completion percentage

## Personal Information

### Basic Details

**Editable Information:**
- First Name
- Last Name
- Email Address (requires verification)
- Phone Number
- Date of Birth
- Profile Picture

**How to Update:**
1. Go to your profile page
2. Click "Edit Profile" button
3. Modify the desired fields
4. Click "Save Changes"

### Profile Picture

**Uploading a Picture:**
1. Click the camera icon on your current picture
2. Choose "Upload New Photo"
3. Select an image file (JPG, PNG, GIF)
4. Crop and adjust as needed
5. Click "Save"

**Picture Requirements:**
- Maximum file size: 5MB
- Recommended dimensions: 400x400 pixels
- Supported formats: JPG, PNG, GIF
- Must be appropriate and professional

**Removing Your Picture:**
1. Click the camera icon on your picture
2. Select "Remove Photo"
3. Confirm the removal
4. A default avatar will be used

## Expert Profiles

### Professional Information

**Required Fields:**
- Professional Title
- Years of Experience
- Areas of Expertise
- Professional Summary

**Optional Fields:**
- Company/Organization
- Education Background
- Certifications
- Portfolio/Work Samples
- Professional Website
- Social Media Links

### Expertise Areas

**Adding Expertise:**
1. Go to your profile edit page
2. Find the "Areas of Expertise" section
3. Click "Add Expertise"
4. Select from predefined categories or add custom
5. Set your proficiency level (Beginner, Intermediate, Expert)
6. Save your changes

**Managing Expertise:**
- Reorder by dragging and dropping
- Edit proficiency levels
- Remove outdated expertise
- Add new skills as you develop them

### Professional Verification

**Verification Process:**
1. Submit required documentation
2. Wait for admin review (typically 2-3 business days)
3. Receive verification status update
4. Display verification badge on profile

**Required Documents:**
- Professional license (if applicable)
- Educational certificates
- Work experience verification
- Identity verification

## Privacy and Visibility

### Profile Visibility Options

**Public Profile:**
- Visible to all platform users
- Appears in search results
- Can be found by search engines
- Recommended for experts

**Private Profile:**
- Only visible to connected users
- Limited search visibility
- Not indexed by search engines
- Suitable for customers

**Custom Visibility:**
- Choose which information to display
- Set different visibility for different fields
- Control who can contact you
- Manage search appearance

### Information Sharing

**What Others Can See:**
- Name and profile picture (always visible)
- Professional information (if expert)
- Contact information (based on settings)
- Activity and reviews (based on settings)

**What Remains Private:**
- Email address (unless shared)
- Phone number (unless shared)
- Date of birth
- Account settings
- Payment information

## Managing Multiple Profiles

### Switching Between Roles

If you have both customer and expert accounts:
1. Click your profile menu
2. Select "Switch to [Role]"
3. Your interface will update accordingly
4. Profile information may differ between roles

### Linking Accounts

**Benefits of Linked Accounts:**
- Single login for multiple roles
- Shared basic information
- Unified notification preferences
- Simplified account management

**How to Link:**
1. Contact support to request account linking
2. Provide both account email addresses
3. Verify ownership of both accounts
4. Support will merge the accounts

## Troubleshooting Profile Issues

### Common Problems

**Can't Update Information:**
- Check for required field validation
- Ensure proper file formats for uploads
- Verify internet connection
- Try refreshing the page

**Profile Picture Won't Upload:**
- Check file size (must be under 5MB)
- Verify file format (JPG, PNG, GIF only)
- Try a different browser
- Clear browser cache

**Verification Delays:**
- Check document quality and completeness
- Respond to any admin requests promptly
- Contact support if delayed beyond 5 business days
